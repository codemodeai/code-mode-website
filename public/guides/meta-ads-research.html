01
Install Claude in Chrome
  1. Open Google Chrome on desktop. (Not supported on other Chromium browsers or mobile.)
  2. Go to the Chrome Web Store and search Claude, or open the extension page directly.
  3. Click "Add to Chrome", then confirm "Add extension" in the pop-up.
  4. Sign in with your Claude account when prompted.
  5. Click the puzzle-piece icon in Chrome's toolbar, then click the thumbtack next to "Claude" to pin it.
  6. Click the pinned Claude icon and grant the requested permissions — this opens the side panel.
Requires a paid plan — Pro, Max, Team, or Enterprise. Free accounts can't use browser automation.
One-time setup
02
Connect Claude to the Meta Ads Library
  1. Open a new tab and go to the Meta Ads Library (facebook.com/ads/library).
  2. Click the pinned Claude icon — the side panel opens and reads whatever tab is active.
  3. Click the mode dropdown above the chat box and pick "Ask before acting" for your first run — it's the safer setting on a new site.
  4. Type your request (see Step 3). A "Permission required" prompt will appear for this site.
  5. Choose "Allow this action" for a single pass, or "Always allow actions on this site" if you'll run this often.
Public data — nothing to set up on Meta's side
03
Send the exact search prompt
  1. In the side panel chat box, type the prompt below — swap in your own niche and country.
  2. Claude replies with a plan listing exactly which pages it will visit. Read it.
  3. Click "Approve plan" to let it run.
  4. Claude opens results inside its own colored tab group and works through them one at a time — you can watch live or switch away.
Prompt On the Meta Ads Library, search "shoes" and filter to India, Active ads. Open the first 10 different brands' active ads. For each one, note the hook, the offer, the ad format (video / image / carousel), and the CTA button text.
04
Break it down and implement it
  1. Ask: "Summarize this as a table — brand, offer, hook, format, CTA."
  2. Ask: "Save this to a Google Doc" (or export it as a note) so it doesn't disappear with the chat.
  3. Scan the table for the angle that repeats across 3+ brands — that repetition is the signal, not any single ad.
  4. Write one new ad draft using that repeated hook or offer, adapted to your own brand and price point.
